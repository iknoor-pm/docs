---
title: "Update Lead List"
api: "PUT https://server.smartlead.ai/api/v1/lead-list/{id}"
description: "Update the name of an existing lead list"
---

## Path Parameters

<ParamField path="id" type="number" required>
  The unique ID of the lead list to update
</ParamField>

## Request Body

<ParamField body="listName" type="string" required>
  New name for the lead list
</ParamField>

<RequestExample>

```bash cURL
curl -X PUT "https://server.smartlead.ai/api/v1/lead-list/500?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"listName": "Q1 2025 Enterprise Prospects - Updated"}'
```

</RequestExample>
