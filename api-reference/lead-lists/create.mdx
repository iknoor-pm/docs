---
title: "Create Lead List"
api: "POST https://server.smartlead.ai/api/v1/lead-lists"
description: "Create a new lead list for organizing and managing leads"
---

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Body Parameters

<ParamField body="name" type="string" required>
  Name of the lead list
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/v1/lead-lists?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "Enterprise Prospects Q1"}'
```

</RequestExample>

## Response Fields

<ResponseField name="ok" type="boolean">
  Whether the operation was successful
</ResponseField>

<ResponseField name="data" type="object">
  Created list object with id and name
</ResponseField>

<ResponseExample>

```json 200 - Success
{
  "ok": true,
  "data": {
    "id": 201,
    "name": "Enterprise Prospects Q1",
    "lead_count": 0,
    "created_at": "2025-12-01T10:00:00.000Z"
  }
}
```

</ResponseExample>
