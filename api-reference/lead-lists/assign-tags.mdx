---
title: "Assign Tags to Lead Lists"
api: "POST https://server.smartlead.ai/api/v1/lead-list/assign-tags"
description: "Add or remove tags from one or more lead lists"
---

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Request Body

<ParamField body="listIds" type="array" required>
  Array of lead list IDs to tag. 1-10 lists allowed.
</ParamField>

<ParamField body="tagIds" type="array" required>
  Array of tag IDs to assign. 1-10 tags allowed.
</ParamField>

<ParamField body="removeTagIds" type="array">
  Array of tag IDs to remove. 1-10 tags allowed.
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/v1/lead-list/assign-tags?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"listIds": [500, 501], "tagIds": [1, 2], "removeTagIds": [3]}'
```

</RequestExample>
