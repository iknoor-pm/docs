---
title: "Get All Tags"
api: "GET https://server.smartlead.ai/api/v1/leads/tags"
description: "Retrieve all available lead tags"
---

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/leads/tags?api_key=YOUR_KEY"
```

</RequestExample>

## Response Fields

<ResponseField name="ok" type="boolean">
  Whether the operation was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of tag objects
</ResponseField>

<ResponseExample>

```json 200 - Success
{
  "ok": true,
  "data": [
    {"id": 101, "name": "Hot Lead", "color": "#FF5733"},
    {"id": 102, "name": "Qualified", "color": "#28A745"}
  ]
}
```

</ResponseExample>
