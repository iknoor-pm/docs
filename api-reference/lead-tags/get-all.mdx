---
title: "Get Lead Tags"
api: "GET https://server.smartlead.ai/api/v1/crm/leads/tags"
description: "Get tags associated with a specific lead"
---

## Query Parameters

<ParamField query="leadId" type="number">
  The lead ID to retrieve tags for. If omitted, returns all available tags.
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/crm/leads/tags?api_key=YOUR_KEY&leadId=123"
```

</RequestExample>

## Response Codes
