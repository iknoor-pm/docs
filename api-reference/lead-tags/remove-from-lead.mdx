---
title: "Remove Tag from Lead"
api: "DELETE https://server.smartlead.ai/api/v1/crm/leads/tags/{tagMappingId}"
description: "Remove a specific tag mapping from a lead"
---

## Path Parameters

<ParamField path="tagMappingId" type="number" required>
  The tag mapping ID (not the tag ID). Get this from the Get Lead Tags response.
</ParamField>

<RequestExample>

```bash cURL
curl -X DELETE "https://server.smartlead.ai/api/v1/crm/leads/tags/789?api_key=YOUR_KEY"
```

</RequestExample>

## Response Codes

<ResponseField name="200" type="Success">

</ResponseField>

<ResponseField name="400" type="Bad Request">
  Invalid request parameters or malformed request body
</ResponseField>

<ResponseField name="401" type="Unauthorized">
