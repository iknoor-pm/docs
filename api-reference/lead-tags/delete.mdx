---
title: "Delete Tag"
api: "DELETE https://server.smartlead.ai/api/v1/leads/tags/{tag_id}"
description: "Permanently delete a tag from the system"
---

## Path Parameters

<ParamField path="tag_id" type="number" required>
  The unique identifier of the tag to delete
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

<RequestExample>

```bash cURL
curl -X DELETE "https://server.smartlead.ai/api/v1/leads/tags/101?api_key=YOUR_KEY"
```

</RequestExample>

## Response Fields

<ResponseField name="ok" type="boolean">
  Whether the operation was successful
</ResponseField>

<ResponseExample>

```json 200 - Success
{
  "ok": true
}
```

</ResponseExample>
