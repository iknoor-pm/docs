---
title: "Update Lead Note"
api: "PATCH https://server.smartlead.ai/api/v1/leads/{lead_id}/notes/{note_id}"
description: "Update an existing note for a lead"
---

## Path Parameters

<ParamField path="lead_id" type="number" required>
  The unique identifier of the lead
</ParamField>

<ParamField path="note_id" type="number" required>
  The unique identifier of the note
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Body Parameters

<ParamField body="note" type="string" required>
  Updated note content
</ParamField>

<RequestExample>

```bash cURL
curl -X PATCH "https://server.smartlead.ai/api/v1/leads/123/notes/456?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"note": "Updated note content"}'
```

</RequestExample>

## Response Fields

<ResponseField name="ok" type="boolean">
  Whether the operation was successful
</ResponseField>

<ResponseField name="data" type="object">
  Updated note object
</ResponseField>
