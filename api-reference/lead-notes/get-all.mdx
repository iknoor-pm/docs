---
title: "Get Lead Notes"
api: "GET https://server.smartlead.ai/api/v1/crm/leads/notes/{id}"
description: "Retrieve all notes for a specific lead"
---

## Path Parameters

<ParamField path="id" type="number" required>
  The lead ID to retrieve notes for
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/crm/leads/notes/123?api_key=YOUR_KEY"
```

</RequestExample>
