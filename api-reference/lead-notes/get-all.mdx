---
title: "Get Lead Notes"
api: "GET https://server.smartlead.ai/api/v1/leads/{lead_id}/notes"
description: "Retrieve all notes for a specific lead"
---

## Path Parameters

<ParamField path="lead_id" type="number" required>
  The unique identifier of the lead
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/leads/123/notes?api_key=YOUR_KEY"
```

</RequestExample>

## Response Fields

<ResponseField name="ok" type="boolean">
  Whether the operation was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of note objects
</ResponseField>

<ResponseExample>

{
