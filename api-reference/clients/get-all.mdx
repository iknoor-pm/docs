---
title: "Get All Clients"
api: "GET https://server.smartlead.ai/api/v1/client/"
description: "Retrieve all client sub-accounts under your main account"
---

<Note>
Returns a list of all whitelabel client sub-accounts. Use this to manage and monitor your agency clients.
</Note>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/client/?api_key=YOUR_KEY"
```

</RequestExample>
