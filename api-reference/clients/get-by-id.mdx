---
title: "Get Client by ID"
api: "GET https://server.smartlead.ai/api/v1/clients/{client_id}"
description: "Retrieve details of a specific client"
---

## Path Parameters

<ParamField path="client_id" type="number" required>
  The unique identifier of the client
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/clients/301?api_key=YOUR_KEY"
```

</RequestExample>

## Response Fields

<ResponseField name="ok" type="boolean">
  Whether the operation was successful
</ResponseField>

<ResponseField name="data" type="object">
  Client details
</ResponseField>
