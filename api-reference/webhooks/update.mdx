---
title: "Add / Update Campaign Webhook"
api: "POST https://server.smartlead.ai/api/v1/campaigns/{campaign_id}/webhooks"
description: "Add a new webhook or update an existing webhook for a campaign"
---

<Note>
Use this endpoint to add a new webhook or update an existing one. To create a new webhook, omit the `id` field. To update an existing webhook, include the webhook `id`.
</Note>

## Path Parameters

<ParamField path="campaign_id" type="number" required>
  The campaign ID to add/update the webhook for
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Request Body

<ParamField body="id" type="number">
  Webhook ID. Omit or set to null to create a new webhook. Include to update an existing webhook.
</ParamField>

<ParamField body="name" type="string" required>
  A descriptive name for the webhook
</ParamField>

<ParamField body="webhook_url" type="string" required>
  The URL to receive webhook events
</ParamField>

<ParamField body="event_types" type="array" required>
  Array of event types to subscribe to (e.g. `EMAIL_SENT`, `EMAIL_OPENED`, `EMAIL_REPLIED`, `EMAIL_CLICKED`, `LEAD_UNSUBSCRIBED`, `EMAIL_BOUNCED`)
</ParamField>

<ParamField body="categories" type="array">
  Array of lead categories to filter events by
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/v1/campaigns/{campaign_id}/webhooks?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My CRM Webhook",
    "webhook_url": "https://example.com/webhook",
    "event_types": ["EMAIL_SENT", "EMAIL_REPLIED", "EMAIL_BOUNCED"],
    "categories": []
  }'
```

```python Python
import requests

API_KEY = "YOUR_API_KEY"

response = requests.post(
    "https://server.smartlead.ai/api/v1/campaigns/{campaign_id}/webhooks",
    params={"api_key": API_KEY},
    json={
        "name": "My CRM Webhook",
        "webhook_url": "https://example.com/webhook",
        "event_types": ["EMAIL_SENT", "EMAIL_REPLIED", "EMAIL_BOUNCED"],
        "categories": []
    }
)

result = response.json()
print(result)
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';

const response = await fetch(
  `https://server.smartlead.ai/api/v1/campaigns/${campaignId}/webhooks?api_key=${API_KEY}`,
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      name: 'My CRM Webhook',
      webhook_url: 'https://example.com/webhook',
      event_types: ['EMAIL_SENT', 'EMAIL_REPLIED', 'EMAIL_BOUNCED'],
      categories: []
    })
  }
);

const result = await response.json();
console.log(result);
```

</RequestExample>

## Response Codes

<ResponseField name="200" type="Success">
  Webhook added/updated successfully
</ResponseField>

<ResponseField name="400" type="Bad Request">
  Invalid request parameters
</ResponseField>

<ResponseField name="401" type="Unauthorized">
  Invalid or missing API key
</ResponseField>

<ResponseField name="404" type="Not Found">
  Campaign not found
</ResponseField>

<ResponseExample>

```json 200 - Success

  "id": 12345,

```

