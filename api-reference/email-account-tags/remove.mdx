---
title: "Remove Tags from Email Accounts"
api: "DELETE https://server.smartlead.ai/api/v1/email-accounts/tag-mapping"
description: "Remove tags from one or more email accounts"
---

<Note>
Remove tag associations from email accounts. This does not delete the tags themselves.
</Note>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Request Body

<ParamField body="email_account_ids" type="array" required>
  Array of email account IDs to remove tags from. Minimum 1 required.
</ParamField>

<ParamField body="tag_ids" type="array" required>
  Array of tag IDs to remove. Minimum 1 required.
</ParamField>

<RequestExample>

```bash cURL
curl -X DELETE "https://server.smartlead.ai/api/v1/email-accounts/tag-mapping?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"email_account_ids": [101, 102], "tag_ids": [1]}'
```

</RequestExample>
