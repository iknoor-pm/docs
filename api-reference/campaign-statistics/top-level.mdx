## Path Parameters

<ParamField path="campaign_id" type="integer" required>
  Campaign ID
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your API key
</ParamField>



<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/campaigns/12345/analytics?api_key=YOUR_API_KEY"
```

```python Python
import requests

API_KEY = "YOUR_API_KEY"
CAMPAIGN_ID = 12345

response = requests.get(
    f"https://server.smartlead.ai/api/v1/campaigns/{CAMPAIGN_ID}/analytics",
    params={"api_key": API_KEY}
)

result = response.json()
print(result)
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';
const CAMPAIGN_ID = 12345;

const response = await fetch(
  `https://server.smartlead.ai/api/v1/campaigns/${CAMPAIGN_ID}/analytics?api_key=${API_KEY}`
);

const result = await response.json();
console.log(result);
```
