---
title: "Fetch Campaign Lead Statistics"
api: "GET https://server.smartlead.ai/api/v1/campaigns/{campaign_id}/leads-statistics"
description: "Fetch campaign lead statistics using the campaign's ID"
---

## Path Parameters

<ParamField path="campaign_id" type="integer" required>
  The ID of the campaign
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your API key
</ParamField>

<ParamField query="limit" type="integer">
  The number of leads you want to fetch in one go (optional, max 100)
</ParamField>

<ParamField query="event_time_gt" type="string">
  Replied/Sent at date in YYYY-MM-DD format. If you want to filter by when the last event for the lead was received by. This can be a reply event, send event etc (optional)
</ParamField>

<ParamField query="offset" type="integer">
  Used to paginate lead data (optional)
</ParamField>



<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/campaigns/12345/leads-statistics?api_key=YOUR_API_KEY&limit=100&offset=1"
```

```python Python
import requests

API_KEY = "YOUR_API_KEY"
CAMPAIGN_ID = 12345

response = requests.get(
    f"https://server.smartlead.ai/api/v1/campaigns/{CAMPAIGN_ID}/leads-statistics",
    params={
        "api_key": API_KEY,
        "limit": 100,
        "offset": 1
    }
)

result = response.json()
print(result)
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';
const CAMPAIGN_ID = 12345;

const response = await fetch(
  `https://server.smartlead.ai/api/v1/campaigns/${CAMPAIGN_ID}/leads-statistics?api_key=${API_KEY}&limit=100&offset=1`
);

const result = await response.json();
console.log(result);
```
