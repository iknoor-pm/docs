---
title: "Fetch Campaign Mailbox Statistics"
api: "GET https://server.smartlead.ai/api/v1/campaigns/{campaign_id}/mailbox-statistics"
description: "Fetch mailbox statistics specific to a campaign"
---

## Path Parameters

<ParamField path="campaign_id" type="integer" required>
  The ID of the campaign
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your API key
</ParamField>

<ParamField query="client_id" type="string" default="1">
  The ID of your client if this campaign is client specific
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Pagination offset
</ParamField>

<ParamField query="limit" type="integer" default="5">
  Number of results to return. Min 1, max 20.
</ParamField>

<ParamField query="start_date" type="date">
  Start date for filtering. Both start_date and end_date must be provided, otherwise data will be for the full campaign length.
</ParamField>

<ParamField query="end_date" type="date">
  End date for filtering. Both start_date and end_date must be provided, otherwise data will be for the full campaign length.
</ParamField>

<ParamField query="time_zone" type="string" default="UTC">
  The campaign timezone, e.g. `America/Los_Angeles`. Same format as shown in your campaign UI next to the date filter.
</ParamField>

## Headers

<ParamField header="accept" type="string" default="application/json">
  Response content type. Allowed: `application/json`, `text/plain`
</ParamField>



<RequestExample>


```python Python
import requests

API_KEY = "YOUR_API_KEY"
CAMPAIGN_ID = 12345

response = requests.get(
    f"https://server.smartlead.ai/api/v1/campaigns/{CAMPAIGN_ID}/mailbox-statistics",
    params={
        "api_key": API_KEY,
        "limit": 10,
        "offset": 0
    }
)

result = response.json()
print(result)
```
