## Path Parameters

<ParamField path="campaign_id" type="integer" required>
  The ID of the campaign
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your API key
</ParamField>

<ParamField query="start_date" type="string" default="2022-12-16">
  Start date in YYYY-MM-DD format
</ParamField>

<ParamField query="end_date" type="string" default="2022-12-23">
  End date in YYYY-MM-DD format
</ParamField>

## Headers

<ParamField header="accept" type="string" default="application/json">
  Response content type. Allowed: `application/json`, `text/plain`
</ParamField>



<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/campaigns/12345/analytics-by-date?api_key=YOUR_API_KEY&start_date=2024-01-01&end_date=2024-01-31"
```

```python Python
import requests

API_KEY = "YOUR_API_KEY"
CAMPAIGN_ID = 12345

response = requests.get(
    f"https://server.smartlead.ai/api/v1/campaigns/{CAMPAIGN_ID}/analytics-by-date",
    params={
        "api_key": API_KEY,
        "start_date": "2024-01-01",
        "end_date": "2024-01-31"
    }
)

result = response.json()
print(result)
```
