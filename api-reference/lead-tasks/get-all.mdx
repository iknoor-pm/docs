---

## Path Parameters

<ParamField path="id" type="number" required>
  The lead ID to retrieve tasks for
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/crm/leads/tasks/123?api_key=YOUR_KEY"
```

</RequestExample>
