<RequestExample>


```python Python
import requests

API_KEY = "YOUR_API_KEY"

response = requests.get(
    "https://server.smartlead.ai/api/v1/analytics/campaign/overall-stats",
    params={"api_key": API_KEY}
)

result = response.json()
print(result)
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';

console.log(result);
```
