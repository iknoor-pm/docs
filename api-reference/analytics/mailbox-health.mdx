## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

<ParamField query="client_ids" type="string">
  Comma-separated client IDs
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/analytics/mailbox/overall-stats?api_key=YOUR_KEY"
```

```python Python
import requests

API_KEY = "YOUR_API_KEY"

response = requests.get(
    "https://server.smartlead.ai/api/v1/analytics/mailbox/overall-stats",
    params={"api_key": API_KEY}
)

result = response.json()
print(result)
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';

const response = await fetch(
  `https://server.smartlead.ai/api/v1/analytics/mailbox/overall-stats?api_key=${API_KEY}`
);

const result = await response.json();
console.log(result);
```

</RequestExample>
