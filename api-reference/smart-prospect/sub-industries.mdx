description: "List sub-industries for the authenticated user - GET /sub-industries"
---

<Note>
This endpoint requires authentication via API key passed as a query parameter (`api_key`).
</Note>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key for authentication
</ParamField>

<ParamField query="limit" type="string">
  Number of sub-industries to return (1–100). Pattern: ^[1-9][0-9]*$ or 100. Default: "10".
</ParamField>

<ParamField query="offset" type="string">
  Number of sub-industries to skip for pagination (≥ 0). Pattern: ^[0-9]+$. Default: "0".
</ParamField>

<ParamField query="search" type="string">
  Search string to match sub-industry names starting with this value (1–255 characters).
</ParamField>

<ParamField query="industry_id" type="string">
  Filter by industry ID (positive integer string).
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://prospect-api.smartlead.ai/api/v1/search-email-leads/sub-industries?api_key=YOUR_API_KEY"
```

</RequestExample>
