---
title: "Get Enrichment Status"
api: "GET https://server.smartlead.ai/api/v1/smart-prospect/enrich/{job_id}/status"
description: "Check the status of a bulk enrichment job"
---

## Path Parameters

<ParamField path="job_id" type="string" required>
  The enrichment job ID returned from bulk enrich
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/smart-prospect/enrich/enr_abc123/status?api_key=YOUR_KEY"
```

</RequestExample>

<ResponseExample>

```json 200 - Success
{
  "ok": true,
  "data": {
    "job_id": "enr_abc123",
    "status": "completed",
    "total": 2,
    "enriched": 2,
    "failed": 0
  }
}
```

</ResponseExample>
