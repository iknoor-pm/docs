---
title: "Get Smart Prospect List Leads"
api: "GET https://server.smartlead.ai/api/v1/smart-prospect/lists/{list_id}/leads"
description: "Retrieve leads from a specific Smart Prospect list"
---

## Path Parameters

<ParamField path="list_id" type="number" required>
  The Smart Prospect list ID
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

<ParamField query="limit" type="number">
  Number of leads to return (default: 25)
</ParamField>

<ParamField query="offset" type="number">
  Pagination offset
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/smart-prospect/lists/301/leads?api_key=YOUR_KEY&limit=25"
```

</RequestExample>
