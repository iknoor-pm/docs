---
title: "Save Search"
api: "POST https://server.smartlead.ai/api/v1/smart-prospect/searches"
description: "Save a prospect search for future reference"
---

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Body Parameters

<ParamField body="name" type="string" required>
  Name for the saved search
</ParamField>

<ParamField body="filters" type="object" required>
  Search filter criteria
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/v1/smart-prospect/searches?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "CEO Search - SF", "filters": {"job_titles": ["CEO"], "locations": ["San Francisco"]}}'
```

</RequestExample>

<ResponseExample>

```json 200 - Success
{
  "ok": true,
  "data": {
    "id": 100,
    "name": "CEO Search - SF",
    "created_at": "2025-12-01T10:00:00.000Z"
  }
}
```

</ResponseExample>
