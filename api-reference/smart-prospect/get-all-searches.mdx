## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/smart-prospect/searches?api_key=YOUR_KEY"
```

</RequestExample>

<ResponseExample>

```json 200 - Success
{
  "ok": true,
  "data": [
    {"id": 100, "name": "CEO Search - SF", "total_results": 150},
    {"id": 101, "name": "Marketing VPs", "total_results": 85}
  ]
}
```

</ResponseExample>
