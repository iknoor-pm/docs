---
title: "Get Search by ID"
api: "GET https://server.smartlead.ai/api/v1/smart-prospect/searches/{search_id}"
description: "Retrieve details and results of a specific saved search"
---

## Path Parameters

<ParamField path="search_id" type="number" required>
  The unique identifier of the saved search
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/smart-prospect/searches/100?api_key=YOUR_KEY"
```

</RequestExample>

<ResponseExample>

```json 200 - Success
{
  "ok": true,
  "data": {
    "id": 100,
    "name": "CEO Search - SF",
    "filters": {"job_titles": ["CEO"], "locations": ["San Francisco"]},
    "total_results": 150,
    "created_at": "2025-12-01T10:00:00.000Z"
  }
}
```

</ResponseExample>
