---
title: "Search Companies"
api: "POST https://server.smartlead.ai/api/v1/smart-prospect/search/companies"
description: "Search for companies using filters like industry, size, location, and technology"
---

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Body Parameters

<ParamField body="industries" type="array">
  Array of industry names
</ParamField>

<ParamField body="locations" type="array">
  Array of locations
</ParamField>

<ParamField body="company_size" type="string">
  Company size range
</ParamField>

<ParamField body="technologies" type="array">
  Array of technologies used
</ParamField>

<ParamField body="limit" type="number">
  Number of results (default: 25, max: 100)
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/v1/smart-prospect/search/companies?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"industries": ["SaaS"], "company_size": "51-200", "limit": 25}'
```

</RequestExample>

<ResponseExample>

```json 200 - Success
{
  "ok": true,
  "data": {
    "total": 80,
    "results": [
      {
        "id": 2001,
        "name": "TechCo",
        "domain": "techco.com",
        "industry": "SaaS",
        "size": "51-200",
        "location": "San Francisco, CA"
      }
    ]
  }
}
```

</ResponseExample>
