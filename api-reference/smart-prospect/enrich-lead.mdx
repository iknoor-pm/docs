---
title: "Enrich Lead"
api: "POST https://server.smartlead.ai/api/v1/smart-prospect/enrich"
description: "Enrich a single lead with additional data (email, phone, company info)"
---

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Body Parameters

<ParamField body="email" type="string">
  Lead email to enrich
</ParamField>

<ParamField body="linkedin_url" type="string">
  LinkedIn profile URL
</ParamField>

<ParamField body="first_name" type="string">
  First name (used with company for lookup)
</ParamField>

<ParamField body="last_name" type="string">
  Last name (used with company for lookup)
</ParamField>

<ParamField body="company" type="string">
  Company name or domain
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/v1/smart-prospect/enrich?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"email": "jane@techco.com"}'
```

</RequestExample>

<ResponseExample>

```json 200 - Success
{
  "ok": true,
  "data": {
    "email": "jane@techco.com",
    "first_name": "Jane",
    "last_name": "Smith",
    "title": "CEO",
    "company": "TechCo",
    "phone": "+1234567890",
    "linkedin_url": "https://linkedin.com/in/janesmith",
    "credits_used": 1
  }
}
```

</ResponseExample>
