---
title: "Head Counts API"
api: "GET https://prospect-api.smartlead.ai/api/v1/search-email-leads/head-counts"
description: "List head counts (company size ranges) for the authenticated user - GET /head-counts"
---

<Note>
This endpoint requires authentication via API key passed as a query parameter (`api_key`).
</Note>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key for authentication
</ParamField>

<ParamField query="limit" type="string">
  Number of head counts to return (1–100). Pattern: ^[1-9][0-9]*$ or 100. Default: "10".
</ParamField>

<ParamField query="offset" type="string">
  Number of head counts to skip for pagination (≥ 0). Pattern: ^[0-9]+$. Default: "0".
</ParamField>

<ParamField query="search" type="string">
  Search string to match head count values starting with this value (1–255 characters).
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://prospect-api.smartlead.ai/api/v1/search-email-leads/head-counts?api_key=YOUR_API_KEY"
```

</RequestExample>
