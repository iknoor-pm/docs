---
title: "Search People"
api: "POST https://server.smartlead.ai/api/v1/smart-prospect/search/people"
description: "Search for people/prospects using various filters like job title, company, location, and industry"
---

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Body Parameters

<ParamField body="job_titles" type="array">
  Array of job titles to search for
</ParamField>

<ParamField body="companies" type="array">
  Array of company names or domains
</ParamField>

<ParamField body="locations" type="array">
  Array of locations (city, state, or country)
</ParamField>

<ParamField body="industries" type="array">
  Array of industry names
</ParamField>

<ParamField body="company_size" type="string">
  Company size range (e.g., `1-10`, `11-50`, `51-200`, `201-500`, `501-1000`, `1001+`)
</ParamField>

<ParamField body="limit" type="number">
  Number of results to return (default: 25, max: 100)
</ParamField>

<ParamField body="offset" type="number">
  Pagination offset
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/v1/smart-prospect/search/people?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"job_titles": ["CEO", "CTO"], "locations": ["San Francisco"], "limit": 25}'
```

</RequestExample>
