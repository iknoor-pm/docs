## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

<RequestExample>

```bash cURL
curl "https://server.smartlead.ai/api/v1/smart-prospect/lists?api_key=YOUR_KEY"
```

</RequestExample>

<ResponseExample>

```json 200 - Success
{

  "data": [
    {"id": 301, "name": "Tech CEOs", "lead_count": 150},
    {"id": 302, "name": "Marketing VPs", "lead_count": 85}
  ]
}
```

</ResponseExample>
