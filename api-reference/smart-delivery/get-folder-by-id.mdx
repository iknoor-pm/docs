## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>



<RequestExample>

```bash cURL
curl "https://smartdelivery.smartlead.ai/api/v1/smart-delivery/folders/{folder_id}?api_key=YOUR_KEY"
```

```python Python
import requests

API_KEY = "YOUR_API_KEY"

response = requests.get(
    "https://smartdelivery.smartlead.ai/api/v1/smart-delivery/folders/{folder_id}",
    params={"api_key": API_KEY}
)

result = response.json()
print(result)
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';

const response = await fetch(
  `https://smartdelivery.smartlead.ai/api/v1/smart-delivery/folders/${folder_id}?api_key=${API_KEY}`
);

const result = await response.json();
console.log(result);
```

</RequestExample>
